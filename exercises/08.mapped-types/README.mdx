# Mapped Types

Mapped types let you create new types by transforming each property of an
existing type. This is how TypeScript's built-in utility types work!

```ts
// Create a type where all properties are optional
type Optional<T> = {
	[K in keyof T]?: T[K]
}

// Create a type where all properties are readonly
type Immutable<T> = {
	readonly [K in keyof T]: T[K]
}
```

## The Syntax

```ts
type MappedType<T> = {
	[K in keyof T]: TransformedType
}
```

- `K in keyof T` iterates over each key of T
- `T[K]` accesses the type of each property
- You can add modifiers like `?`, `readonly`, or `-readonly`

## How Built-in Utilities Work

```ts
// Partial<T> is just:
type Partial<T> = { [K in keyof T]?: T[K] }

// Required<T> removes the optional modifier:
type Required<T> = { [K in keyof T]-?: T[K] }

// Readonly<T>:
type Readonly<T> = { readonly [K in keyof T]: T[K] }
```
