# Typing Async Code

TypeScript's type system works beautifully with async code. When you mark a
function as `async`, TypeScript automatically infers that it returns a Promise.

```ts
// TypeScript infers: () => Promise<string>
async function getName() {
	return 'Alice'
}

// Explicit type annotation
async function getName(): Promise<string> {
	return 'Alice'
}
```

## Type Inference with Async

- **Return types** - `async` functions always return `Promise<T>`
- **Await types** - `await` unwraps the Promise type
- **Error types** - TypeScript helps catch type errors in catch blocks

<callout-info>
Understanding how TypeScript types async code helps you write safer code
and catch errors at compile time rather than runtime.
</callout-info>

In this exercise, you'll properly type async functions and understand how
TypeScript handles Promise types.
